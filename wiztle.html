WizTLE: Minimal Multi-Satellite Tracking Software for XY and AZEL Antenna Mounts


WizTLE v1.0-alpha is a command line executable for scheduling and tracking multiple satellites using arguments inputted on program execution using the terminal. After execution WizTLE will perform the necessary operation and terminate when it's done. The format of position commands are configured to work with Profen's antenna control systems. In TwinCAT environments PLC will trigger WizTLE using NT_StartProcess function block while also giving it command line arguments in COMNDLINE input.
